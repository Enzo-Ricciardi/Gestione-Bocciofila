# Gestione Bocciofila Roadmap

## Objective

Prepare Gestione Bocciofila for open collaboration while preserving a clean, deployable, club-oriented web application.

## Priority areas

1. Reduce repository noise and historical bundle clutter
2. Clarify the source/build workflow
3. Improve contributor onboarding
4. Harden deployment and operational hygiene
5. Review data import/export and draw reliability

## Workstreams

### Repository hygiene

- remove obsolete bundles from future revisions
- keep operational artifacts out of version control
- document the current production bundle clearly

### Product and workflow

- review draw generation robustness
- improve player import/export safety
- improve print/export workflows

### Deployment

- preserve environment-based FTP configuration
- document safe release steps
- keep diagnostics clearly separated from production behavior
